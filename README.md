### Usage

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: nginx
spec:
  containers:
  - image: nginx:${NGINX_VERSION}
    name: nginx
    ports:
    - containerPort: ${PORT | default(80)}
    restartPolicy: Never
```

```bash
NGINX_VERSION=1.23.4 envsub < manifest.yaml
```
